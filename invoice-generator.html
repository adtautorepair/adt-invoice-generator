Invoice wording

Format is PREFIX-YY-COUNTER (e.g. INV-26-1001). The year updates automatically, and the counter goes up by 1 each time you save an invoice — sequential and gap-free, the way the CRA expects.

Email account (SMTP)

Connect your Gmail so invoices send straight from the app (no external mail program). Gmail needs an App Password (Google Account → Security → 2-Step Verification → App passwords), not your normal password.
Placeholders {name} {invoice} {total} {company} {phone} auto-fill per customer.
Note: connecting/sending over SMTP runs in the installable desktop version of this app (a browser file cannot send email directly). The settings and composer are ready now; sending switches on when we package the desktop app.

AI note helper (optional)

Lets the Improve writing button on the Create tab rewrite work notes into clean, professional English — handy if English isn't your first language. Uses Google Gemini's free tier.
Get a free key at aistudio.google.com → “Get API key”. It’s free, used only for the Improve-writing button, and needs internet when used. If you see a “quota/limit” error, wait a few minutes for a new key to activate, or try another model here (e.g. gemini-2.0-flash or gemini-flash-latest).
